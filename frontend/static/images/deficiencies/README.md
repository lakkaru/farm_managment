# Rice Plant Nutritional Deficiency Images

This directory contains reference images for rice plant nutritional deficiencies.

## Image Sources

All images are sourced from the **IRRI Rice Knowledge Bank** (International Rice Research Institute) - a trusted global authority on rice cultivation.

## Image Files

- `nitrogen.jpg` - Nitrogen (N) deficiency symptoms
- `potassium.jpg` - Potassium (K) deficiency symptoms  
- `phosphorus.jpg` - Phosphorus (P) deficiency symptoms
- `zinc.jpg` - Zinc (Zn) deficiency symptoms
- `iron-toxicity.jpg` - Iron (Fe) toxicity symptoms
- `sulfur.jpg` - Sulfur (S) deficiency symptoms

## Usage

These images are displayed in the Disease Detection feature at `/paddy/disease-detection` to help farmers identify nutritional deficiencies in their rice crops.
